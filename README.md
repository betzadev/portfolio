## ✨ Características y Stack Tecnológico

- **Arquitectura y Rendimiento**: Desarrollado con **Next.js 16** y **React 19**, utilizando el moderno App Router.
- **Tipado Fuerte**: Código mantenible y escalable gracias a **TypeScript**.
- **Diseño e Interfaz**: Experiencia de usuario (UX) inmersiva, completamente responsiva, usando **Tailwind CSS v4**.
- **Animaciones**: Transiciones fluidas, *scroll parallax* y micro-interacciones manejadas a través de **GSAP** y **Motion**.

## 🚀 Instalación y Ejecución Local

Para correr este proyecto en tu entorno local, asegúrate de tener instalado [Node.js](https://nodejs.org/) y [pnpm](https://pnpm.io/).

1. **Clonar repositorio e instalar dependencias**
   ```bash
   git clone https://github.com/betzadev/portfolio.git
   cd portfolio
   pnpm install
   ```

2. **Iniciar servidor de desarrollo**
   ```bash
   pnpm dev
   ```
   Abre [http://localhost:3000](http://localhost:3000) en tu navegador para interactuar con la aplicación.

3. **Construir para producción**
   ```bash
   pnpm build
   pnpm start
   ```
